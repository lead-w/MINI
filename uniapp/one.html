# 项目配置

  • 服务端和完成的项目导入 hbuilderX

# 配置hbuilderX关联微信开发者工具

  • 1、hbuilderX的运行，配置自己的小程序安装路径
  • 2、在微信开发者工具中开启端口
  • 3、启动项目需要在项目manifest.json配置文件中 小程序配置中 填写自己的appId(注意:每个项目都要重新配置manifest.json文件)

# 搭建服务

  • 1、后端guiguShop_server项目, 提供接口

    • hbuilderX 里面 点击右边,选择使用命令行窗口打开所在目录 => 使用内置终端(进行安装),底部会出现终端,可以输入命令行
    • 运行npm run start开启服务

  • 2、uni.request 用来请求

    • method 有效值必须大写，每个平台支持的method有效值不同，对照官网。
    • data 数据说明,最终发送给服务器的数据是 String 类型，如果传入的 data 不是 String 类型，会被转换成 String。转换规则如下：
    • 对于 GET 方法，例如 { name: 'name', age: 18 } 转换后的结果是 name=name&age=18。
    • 对于 POST 方法且 header['content-type'] 为 application/json 的数据，会进行 JSON 序列化。
uni.request({
    url: 'https://www.example.com/request', //仅为示例，并非真实接口地址。
    data: {
        text: 'uni.request'
    },
    header: {
        'custom-header': 'hello' //自定义请求头信息
    },
    success: (res) => {
        console.log(res.data);
        this.text = 'request success';
    }
});

# 封装请求工具，安装vuex，请求获取首页的数据保存到vuex

  • 1、npm init -y 初始化node工程
  • 2、npm install vuex 安装vuex
  • 一、创建全局 store
    • store/index.js
import Vue from 'vue';
import Vuex from 'vuex';
import home from './modules/home';
Vue.use(Vuex)
const state = {};
const mutations = {};
const actions = {};
const getters = {};

export default new Vuex.Store({
	state,
	getters,
	mutations,
	actions,
	modules:{
		home
	}
})
  • 二、创建home store
    • store/modules/home.js
    • 1、通过actions发送请求,获取首页的数据,保存到store中,后端接口对应的是getHomeData
    • 2、通过mutations方法修改state的数据
import request from '../../utils/request.js'
const state = {
	homeData:{}
};
const mutations = {
	SET_HOMEDATA(state,homeData){
			state.homeData = homeData
	}
};
const actions = {
	async getHomeData({commit}){
		const result = await request('/getHomeData')
		commit('SET_HOMEDATA',result)
	}
};
const getters = {};
export default {
		state,
		mutations,
		actions,
		getters
}
  • 三、封装接口
    • uni.request 用来请求
// utils/config.js
export default{
	host:'http://localhost:3001'
}
// utils/request.js
import config from './config.js'
export default function request(url,data={},method='GET'){
	return new Promise((resolve,reject)=>{
			uni.request({
				url: config.host + url,
				data,
				method,
				success:(res)=>{
					resolve(res)
				},
				fail:(err)=>{
					reject(err)
				}
			})
	})
}

		 return (state.homeData.kingKongModule || {}).kingKongList || []
	 },

		推荐
	</div>
	<div class="item" v-for="(item,index) in navList" :key="item.L1Id" :class="{active:currentIndex === index}" @click="changenav(index)">
		{{item.text}}
	</div>
</scroll-div>
<script>
import {mapGetters} from 'vuex';
export default{
	data(){
		return{
			currentIndex:-1,
		}
