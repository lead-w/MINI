# 7、自定义组件

  • 文档
  • 类似vue, 创建组件
  • 1、创建一个components 目录专门存放组件
  • 2、组件创建 => 在下面创建一个header 组件
    • 组件传值 properties

TIP

控制台报如下问题“已被代码依赖分析忽略，无法被其他模块引用。你可根据控制台中的【代码依赖分析】告警信息修改代码，或关闭【过滤无依赖文件】功能”

处理办法: 1、需要在 开发工具 右上角“详情”=>“本地设置”=>“上传时过滤无依赖文件”关闭即可 2、在project.config.json=>setting里面将ignoreUploadUnusedFiles都设置为 true，然后保存

# 8、封装发请求的功能函数（使用promise需要设置）

  • wx.request
  • 参数
    • url 开发者服务器接口地址
    • data 请求的参数(string/object/ArrayBuffer)
    • header 设置请求的 header，header 中不能设置 Referer。 content-type 默认为 application/json
    • success 成功回调
    • fail 失败回调
    • complete 完成回调
wx.request({
  url: 'example.php', //仅为示例，并非真实的接口地址
  data: {
    x: '',
    y: ''
  },
  header: {
    'content-type': 'application/json' // 默认值
  },
  success (res) {
    console.log(res.data)
  },
  fail (res) {
    console.log(res.data)
  },
  complete (res) {
    console.log(res.data)
  }
})
  • 封装请求函数 创建文件 utils/request.js
export default function request(url, data, method='GET'){
  return new Promise((resolve,rejct)=>{
    wx.request({
      url: 'http://localhost:3000' + url,
      method,
      data,
      success:(res)=>{
        resolve(res.data)
      },
      fail:(res)=>{
        rejct(res)
      }
    })
  })
}
